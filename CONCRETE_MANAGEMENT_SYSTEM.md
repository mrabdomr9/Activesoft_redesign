# نظام إدارة محطات الخرسانة الجاهزة

## نظرة عامة

برنامج إدارة محطات الخرسانة الجاهزة هو منظومة متكاملة للتحكم ومراقبة وإدارة محطات الخرسانة الجاهزة. تم تصميم هذا البرنامج بعناية لتلبية متطلبات مصانع الخرسانة الجاهزة على اختلاف أنواعها من حيث المحتوى والجهد الخرساني.

يوفر النظام مجموعة شاملة من الميزات التي تمكن المستخدمين من:
- مراقبة الجودة والتصنيع
- إدارة السايلوهات والخلاطات والبامب
- تعريف معادلات الإنتاج الخاصة بكل نوع من الخرسانة
- إدارة مواقع العملاء وإنشاء البونات
- إصدار الفواتير اليومية

## المميزات الرئيسية

### 1. إدارة متعددة المحطات
- العمل على أكثر من محطة مع بعضها البعض
- إمكانية تنقل السيارات (مراكز تكلفة) بين المحطات (الفروع)

### 2. تتبع عملية الإنتاج
- متابعة عملية إنتاج الخرسانة من دخول مواد الخام حتى تجهيز وتصنيع الخرسانة
- تنبيه تلقائي لكميات المخزون من مواد الخام

### 3. مرونة في الإنتاج
- ربط المنتج التام بمواد الخام
- إمكانية تغيير أحد المكونات في وقت الإنتاج في حالة نفاذ الكميات أو استبدال صنف بآخر

### 4. البيع المباشر
- إمكانية التصنيع للبيع المباشر - تصنيع كميات وبيعها مباشرة للعميل في خطوة واحدة

### 5. التقارير والتحليلات
- تقارير مفصلة لعمليات الإنتاج ومطابقتها مع تقارير المعمل

### 6. إدارة الفواتير والضرائب
- إمكانية إنشاء فواتير بضريبة مركبة على مستوى الفاتورة وعلى مستوى الصنف

### 7. إدارة التكاليف
- حساب تكلفة وإيراد السيارات والخلاطات والمضخات (مراكز التكلفة) لمعرفة الخسارة والربح تفصيلي
- إمكانية استخراج قائمة دخل وميزانية لكل فرع منفصل
- إمكانية استخراج قائمة دخل وميزانية للشركة مجمعة
- إنشاء عدد لا نهائي من التكاليف المباشرة وغير المباشرة وربطها بالمنتج
- توزيع عادل ودقيق للتكاليف غير المباشرة واحتساب تكلفة المنتج بأدق ما يكون (آلياً)

## الوحدات الرئيسية للنظام

### 1. العملاء والمبيعات
- ربط العملاء بالمندوب
- تعريف الخلاطات والمضخات
- إنشاء البونات للعملاء

### 2. الأصول الثابتة
- تعريف الأصول الثابتة
- تحديد أقساط الإهلاك
- إدخال قيود الإهلاك بين المصروفات ومخصصات الإهلاك

### 3. المخزون والمخازن
- تعريف حسابات المخزون سواء سايلوهات الأسمنت أو مخازن الرمل
- تنفيذ كافة الحركات المخزنية

### 4. أوراق القبض
- عمل قيد تلقائي نتيجة استلام ورقة قبض بين العميل والمنشأة
- تنفيذ جميع الحركات المحاسبية

### 5. أوراق الدفع
- تنفيذ كافة الحركات الخاصة بأوراق الدفع مثل (السداد – الارتداد)
- إنشاء القيود المحاسبية تلقائياً

### 6. حسابات العملاء والمقبوضات
- إدخال المقبوضات سواء كانت نقدية أو بشيك
- توفير تقارير تفصيلية لمتابعة حسابات العملاء

### 7. حسابات الموردين والمدفوعات
- متابعة حسابات الموردين
- استخراج كشوف الحسابات الخاصة بهم

### 8. الحسابات العامة
- دليل الحسابات بعدد لا نهائي من المستويات
- القيود اليومية التي تنشأ تلقائياً من أجزاء النظام المختلفة

### 9. الخزائن والبنوك
- تعريف أكثر من خزينة
- إثبات عمليات القبض والصرف من الخزينة

## الحركة المالية

يوفر النظام العديد من السندات:
- سندات القبض وتفاصيلها (رقم السند والإيصال ورقم القيد والتحليلي الخاص به)
- سندات الدفع وربطها بالحسابات وتاريخ الاستحقاق

يمكن للبرنامج أيضاً:
- تكويد حركات المتحصلات من العملاء والمدفوعات للموردين
- تكويد المندوب القائم بذلك سواء كانت هذه الحركة نقداً أو شيك أو حوالة أو شبكة أو محصل مباشر
- تجميع كل البيانات المتطلبة في نوع الحركة
- تحصيل الشيكات المستحقة آلياً

## التحويل والتسويات والأذونات

يوفر النظام سهولة في:
- التحويل من مخزن لمخزن
- طلب التحويل
- تسويات الصرف والإضافة والتوالف
- تسوية الفيزا
- تسوية شيكات المبيعات
- أذونات الصرف والإضافة ومرتجعاتهم

## تسجيل الإنتاج

- تعريف معادلات الخرسانة طبقاً للجهود المختلفة
- بمجرد عمل البون يقوم البرنامج بعملية الإنتاج تلقائياً

## شؤون الموظفين والرواتب

- إنشاء قاعدة بيانات للموظفين
- تعريف الوظائف المختلفة
- إرفاق المستندات الخاصة بالموظفين

## الموردين والمشتريات

- إنشاء أوامر الشراء وأوامر التوريد
- إنشاء فواتير المشتريات
- إنشاء قاعدة بيانات للموردين بكافة بياناتهم

## إجراءات المحاسبة

- ترحيل للحسابات والمراكز الفرعية لمجموعة قيود وبشروط معينة
- إمكانية إعادة ترقيم القيود اليومية

## الأصناف

يمكن للبرنامج إدارة المستودعات والمخازن والأصناف من خلال:
- بيانات شاملة للأصناف (الاسم والكود وأكواد بديلة للصنف وأصناف بديلة وباركود الكرتونة ووحدات قياس للأصناف)
- تكويد الأصناف في مجموعات
- تحديد ما إذا كان الصنف عادي أو خلطة
- في حالة الخلطة، تحديد المواد الخام الداخلة في تكوينها
- عند تركيب الخلطة، خصم تلقائي من المواد الخام حسب نسبتها في تكوين الخلطة
- إمكانية إضافة خدمة على الصنف وسعر لكل خدمة (خدمة التوصيل مثلاً)
- إضافة صور للأصناف
- التكويد في أكثر من مخزن بأسعار مختلفة
- أنواع خدمات الأصناف والعمولة الواردة عليها
- تجميعي الأصناف

## Technical Implementation

This system is built as part of our React/TypeScript web application using modern development practices:

- **Frontend Framework**: React with TypeScript
- **Routing**: React Router for navigation
- **State Management**: Context API for language and global state
- **Styling**: Tailwind CSS for responsive design
- **Build Tool**: Vite for fast development and production builds

### Key Components

1. **ConcreteManagementPage.tsx** - Main page component for the concrete management system
2. **LanguageContext.tsx** - Handles Arabic/English language switching
3. **useTranslation.ts** - Custom hook for accessing translations
4. **Header.tsx / Footer.tsx** - Navigation and footer components
5. **FloatingCTA.tsx** - Call-to-action component

### Translation System

The system supports both Arabic and English languages through JSON translation files:
- **ar.json** - Arabic translations
- **en.json** - English translations

All content is fully localized and the UI adapts to right-to-left (RTL) for Arabic and left-to-right (LTR) for English.

## Deployment

To run the system locally:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```

The system is designed to be easily deployable to any static hosting service like Vercel, Netlify, or traditional web servers.
